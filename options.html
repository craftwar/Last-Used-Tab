Keyboard shortcuts must include exactly one of Control (specify Ctrl on windows, MacCtrl on Apple), Alt or ⌘Command (specify as Ctrl on Mac) keys. They may optionally include Shift. The key can be any of:

  • the letters A-Z
  • the numbers 0-9
  • the function keys F1-F12
  • Comma, Period, Home, End, PageUp, PageDown, Space, Insert, Delete, Up, Down, Left, Right

The command should be in the format Ctrl+Shift+1 (the default). Letters must be capitalized.

If a key combination is already used by the browser (for example, "Ctrl+P"), or by an existing add-on, then you can't override it. You will be allowed to define it, it will not trigger this addon. Unfortunately there is no way for this addon to detect that.

On Mac computers, "Ctrl" in the shortcut above means "⌘ Command" due to the cross-platform nature of Firefox. If you want control on mac, use "MacCtrl"
